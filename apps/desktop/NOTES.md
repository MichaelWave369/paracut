# ParaCut Desktop

This folder is reserved for the first ParaCut desktop application.

For v0.2, the goal is to keep the editing model clean before building the interface. The desktop layer should call the project, timeline, media, render, and ledger cores instead of duplicating logic.

## First Interface Milestone

- Media shelf
- Two-track timeline
- Preview placeholder
- Receipt panel
- Export queue panel

The desktop application should stay thin while the core packages do the important work.
